## 🚀 使用建议

### 开发中使用

1. **组件开发**
   ```tsx
   import Image from 'next/image';

   <Image
     src="/icons/tech-hexagon.svg"
     alt="Tech Hexagon"
     width={24}
     height={24}
     className="hover:scale-110 transition-transform"
   />
   ```

2. **CSS 背景**
   ```css
   .cyber-background {
     background-image: url('/decorations/hex-pattern.svg');
     background-repeat: repeat;
   }
   ```

3. **装饰元素**
   ```tsx
   <div className="absolute top-0 left-0">
     <Image src="/decorations/corner-decor.svg" alt="" />
   </div>
   ```

### 设计规范遵循

1. **颜色使用**
   - 使用 CSS 变量保持一致性
   - 遵循 WCAG AA 对比度标准
   - 适度使用发光效果

2. **尺寸规范**
   - 小图标: 16px
   - 标准图标: 24px
   - 大图标: 32-48px
   - 插画: 根据需要

3. **动画使用**
   - 优先使用内置动画
   - 动画时长 2-3秒
   - 避免过度使用
