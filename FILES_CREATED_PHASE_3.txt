## 🚀 使用指南

### 安装依赖
```bash
cd frontend
npm install
```

### 启动开发服务器
```bash
npm run dev
```

### 访问页面
- 通知页面: http://localhost:3000/notifications
- Feed 页面: http://localhost:3000/feed

### API 端点
- 关注 API: http://localhost:3000/api/follow
- 通知 API: http://localhost:3000/api/notifications
- 点赞 API: http://localhost:3000/api/likes
