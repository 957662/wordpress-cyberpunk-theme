🔧 使用示例
================================================================================

1. 导入分享组件:
   import { SocialShareButtons } from '@/components/social';
   
   <SocialShareButtons
     url="https://example.com"
     title="Amazing Post"
     variant="horizontal"
   />

2. 导入用户卡片:
   import { UserCard } from '@/components/social';
   
   <UserCard user={userData} showFollowButton showStats />

3. 导入工具函数:
   import { formatNumber, formatRelativeTime } from '@/lib/utils/social-helpers';
   
   formatNumber(1500); // "1.5K"
   formatRelativeTime(dateString); // "2小时前"

4. 访问页面:
   /feed - 社交动态
   /discover - 发现页面
