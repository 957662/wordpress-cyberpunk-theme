# Cyberpunk WordPress Theme

> 赛博朋克风格的 WordPress 主题

## 主题特点

- 🌃 **深色背景** - 沉浸式赛博朋克氛围
- 💜 **霓虹色彩** - 青色、品红、黄色霓虹效果
- ✨ **发光效果** - 边框和文字发光
- 📺 **扫描线** - CRT 显示器效果
- 📱 **响应式** - 支持桌面、平板、手机

## 文件结构

```
cyberpunk-theme/
├── style.css       # 主样式（赛博朋克风格）
├── functions.php   # 主题功能
├── header.php      # 头部模板
├── footer.php      # 底部模板
├── index.php       # 首页模板
└── single.php      # 文章页模板
```

## 配色方案

| 颜色 | 代码 | 用途 |
|:----:|:----:|:-----|
| 深色背景 | `#0a0a0f` | 主背景 |
| 霓虹青 | `#00f0ff` | 高亮、链接 |
| 霓虹品红 | `#ff00ff` | 强调、按钮 |
| 霓虹黄 | `#f0ff00` | 警告、提示 |

## 安装

1. 下载主题文件
2. 上传到 `/wp-content/themes/cyberpunk/`
3. 在 WordPress 后台启用主题
