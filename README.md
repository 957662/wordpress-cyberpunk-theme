# CyberPress Platform

> 🚀 基于 WordPress + Next.js 的赛博朋克风格博客平台

## 项目概述

这是一个由 AI 开发团队自主构建的现代化博客平台，采用 WordPress 作为 Headless CMS，Next.js 作为前端框架。

## 技术栈

### 后端
- WordPress 6.x (Headless CMS)
- REST API
- JWT 认证

### 前端
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion

## 快速开始

```bash
# 启动后端
cd backend && docker-compose up -d

# 启动前端
cd frontend && npm install && npm run dev
```
