## 🎨 组件属性参考

### PaginationNew
| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| currentPage | number | - | 当前页码（必填） |
| totalPages | number | - | 总页数（必填） |
| onPageChange | (page: number) => void | - | 页码变化回调（必填） |
| totalItems | number | - | 总条目数 |
| pageSize | number | - | 每页条目数 |
| showQuickJump | boolean | false | 显示快速跳转 |
| showPageSelector | boolean | false | 显示页码选择器 |
| showTotal | boolean | false | 显示总数信息 |
| color | 'cyan' \| 'purple' \| ... | 'cyan' | 颜色主题 |
| variant | 'default' \| 'outline' \| ... | 'default' | 样式变体 |

### CarouselNew
| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| items | React.ReactNode[] | - | 轮播项目（必填） |
| autoplay | boolean | true | 自动播放 |
| autoplayInterval | number | 5000 | 播放间隔（毫秒） |
| transition | 'slide' \| 'fade' \| ... | 'slide' | 过渡效果 |
| showNav | boolean | true | 显示导航按钮 |
| showIndicators | boolean | true | 显示指示器 |
| indicatorStyle | 'dots' \| 'lines' \| ... | 'dots' | 指示器样式 |

### CodeHighlighterNew
| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| code | string | - | 代码内容（必填） |
| language | string | 'javascript' | 编程语言 |
| theme | string | 'oneDark' | 主题 |
| showLineNumbers | boolean | true | 显示行号 |
| showCopyButton | boolean | true | 显示复制按钮 |
| showFilename | boolean | true | 显示文件名 |
| expandable | boolean | false | 可展开/收起 |

## 🗄️ 数据库表结构

### 核心表
- **users** - 用户表
- **posts** - 文章表
- **comments** - 评论表
- **categories** - 分类表
- **tags** - 标签表
- **likes** - 点赞表
- **bookmarks** - 收藏表
- **follows** - 关注表
- **reading_progress** - 阅读进度
- **notifications** - 通知表

### 性能表
- **cache_entries** - 缓存表
- **daily_stats** - 每日统计
- **search_history** - 搜索历史
- **post_trends** - 文章趋势
- **user_events** - 用户行为
