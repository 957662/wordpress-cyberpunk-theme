## 🖼️ Logo 设计规范

### Logo 变体

#### 1. 主 Logo (Logo Main)
- **尺寸**: 200x200px (可缩放)
- **格式**: SVG
- **用途**: 网站头部、主要品牌展示
- **文件**: `/public/logo-main.svg`

#### 2. 图标 Logo (Logo Icon)
- **尺寸**: 64x64px (可缩放)
- **格式**: SVG
- **用途**: Favicon、小图标
- **文件**: `/public/logo-favicon.svg`

#### 3. 方形 Logo (Logo Square)
- **尺寸**: 512x512px
- **格式**: SVG + PNG
- **用途**: 社交媒体、应用图标
- **文件**: `/public/logo-square.svg`

### Logo 设计规范
```
┌─────────────────────────────┐
│  ╔═════════════════════╗    │
│  ║       CYBER         ║    │
│  ║       ▼▼▼           ║    │
│  ║      ╔═══╗          ║    │
│  ║      ║ █ ║          ║    │
│  ║      ║█ █║          ║    │
│  ║      ║ █ ║          ║    │
│  ║      ╚═══╝          ║    │
│  ║      PRESS          ║    │
│  ╚═════════════════════╝    │
└─────────────────────────────┘
```

**设计元素**:
- 三角形: 代表科技/进步
- 方形边框: 代表稳定/可靠
- 霓虹青 + 赛博紫渐变
- 故障效果细节
