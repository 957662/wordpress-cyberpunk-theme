## 🎨 配色方案

### 主色调
```css
/* 霓虹青 - 主强调色 */
--cyber-cyan: #00f0ff;
--cyber-cyan-light: #40f5ff;
--cyber-cyan-dark: #00a8b3;
--cyber-cyan-glow: rgba(0, 240, 255, 0.5);

/* 赛博紫 - 次强调色 */
--cyber-purple: #9d00ff;
--cyber-purple-light: #b840ff;
--cyber-purple-dark: #6d00b3;
--cyber-purple-glow: rgba(157, 0, 255, 0.5);

/* 激光粉 - 警告色 */
--cyber-pink: #ff0080;
--cyber-pink-light: #ff4099;
--cyber-pink-dark: #b30059;
--cyber-pink-glow: rgba(255, 0, 128, 0.5);
```

### 中性色
```css
/* 深色背景 */
--bg-primary: #0a0a0f;
--bg-secondary: #16162a;
--bg-tertiary: #1e1e36;
--bg-elevated: #252542;

/* 浅色文本 */
--text-primary: #ffffff;
--text-secondary: #b0b0d0;
--text-tertiary: #8080a0;
--text-muted: #606080;

/* 边框 */
--border-primary: rgba(0, 240, 255, 0.3);
--border-secondary: rgba(157, 0, 255, 0.3);
--border-tertiary: rgba(255, 0, 128, 0.3);
```

### 渐变色
```css
/* 主渐变 */

--gradient-dark: linear-gradient(180deg, #0a0a0f 0%, #16162a 100%);

```

## 🎭 视觉效果

### 发光效果
```css
/* 文字发光 */
.text-glow-cyan {
  text-shadow: 0 0 10px var(--cyber-cyan-glow),
               0 0 20px var(--cyber-cyan-glow),
               0 0 30px var(--cyber-cyan-glow);
}

.text-glow-purple {
  text-shadow: 0 0 10px var(--cyber-purple-glow),
               0 0 20px var(--cyber-purple-glow),
               0 0 30px var(--cyber-purple-glow);
}

/* 边框发光 */
.border-glow-cyan {
  box-shadow: 0 0 5px var(--cyber-cyan-glow),
              0 0 10px var(--cyber-cyan-glow),
              inset 0 0 5px var(--cyber-cyan-glow);
}

/* 按钮发光 */
.button-glow {
  box-shadow: 0 0 15px var(--cyber-cyan-glow),
              0 0 30px var(--cyber-cyan-glow);
}
```

### 卡片效果
```css
.cyber-card {

}
```

## 📐 尺寸规范

- **Medium**: 24px (标准图标)
- **Large**: 32px (强调图标)
- **XLarge**: 48px (英雄区域图标)

### 间距
- **XS**: 4px
- **SM**: 8px
- **MD**: 16px
- **LG**: 24px
- **XL**: 32px

### 圆角
- **Small**: 4px
- **Medium**: 8px
- **Large**: 16px
- **Full**: 9999px (圆形)

## 🎬 动画规范

### 过渡时间
- **Fast**: 150ms (悬停效果)
- **Normal**: 300ms (标准过渡)
- **Slow**: 500ms (页面切换)

### 缓动函数
- **Ease**: `cubic-bezier(0.4, 0, 0.2, 1)`
- **Ease In**: `cubic-bezier(0.4, 0, 1, 1)`
- **Ease Out**: `cubic-bezier(0, 0, 0.2, 1)`
- **Bounce**: `cubic-bezier(0.68, -0.55, 0.265, 1.55)`
