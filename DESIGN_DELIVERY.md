## 🚀 使用方式

### 快速开始

```tsx
// 1. 导入组件
import { Icon, LogoDisplay, Illustration } from '@/components/graphics';

// 2. 使用图标
<Icon name="home" size={24} variant="default" />

// 3. 显示 Logo
<LogoDisplay variant="main" size={200} />

// 4. 展示插画
<Illustration name="cyber-city" fill />
```

### 配置 Tailwind

```typescript
// tailwind.config.ts 已配置
export default {
  theme: {
    extend: {
      colors: {
        cyber: {
          dark: '#0a0a0f',
          cyan: '#00f0ff',
          purple: '#9d00ff',
          pink: '#ff0080',
          yellow: '#f0ff00',
        }
      }
    }
  }
}
```
