## 🚀 快速开始

### 1. 查看示例页面

```bash
cd frontend
npm run dev
```

访问: `http://localhost:3000/examples/cyber-components`

### 2. 导入组件

```tsx
import {
  CyberButton,
  CyberCard,
  CyberInput,
  ParticleBackground,
  CyberChart,
  CyberLoader
} from '@/components/cyber';
```

### 3. 使用组件

```tsx
<CyberCard variant="neon" className="p-6">
  <h2 className="text-2xl text-cyber-cyan mb-4">
    赛博朋克世界
  </h2>
  <CyberInput
    label="用户名"
    variant="glass"
    className="mb-4"
  />
  <CyberButton variant="holographic">
    开始探索
  </CyberButton>
</CyberCard>
```
