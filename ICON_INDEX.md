## 🎨 颜色主题

所有图标支持使用 Tailwind 颜色类或自定义颜色：

### Tailwind 颜色类

```tsx
<HomeIcon size={24} className="text-cyber-cyan" />
<SettingsIcon size={24} className="text-cyber-purple" />
<DeleteIcon size={24} className="text-cyber-pink" />
```

### 自定义颜色

```tsx
<HomeIcon size={24} color="#00f0ff" />
<SettingsIcon size={24} color="rgb(157, 0, 255)" />
```

### 发光效果

```tsx
// SVGIcons.tsx 图标
<HomeIcon size={32} glow={true} />

// CyberIcons.tsx 图标（默认开启）
<NeuralNetworkIcon size={100} glow={false} /> {/* 禁用发光 */}
```

## 🔧 高级用法

### 点击事件

```tsx
<GitHubIcon size={24} onClick={() => window.open('https://github.com')} />
```

### 条件渲染

```tsx
{isLoading ? (
  <LoadingRing size={32} />
) : (
  <CheckIcon size={24} className="text-cyber-green" />
)}
```

### 动画图标

```tsx
// 内置动画
<AudioWaveIcon size={32} />

// 自定义动画类
<HomeIcon size={24} className="animate-pulse" />
```

### 组合使用

```tsx
<div className="relative group">
  <CardIcon size={48} className="text-cyber-cyan" />
  <div className="absolute -top-1 -right-1">
    <NotificationBadge />
  </div>
</div>
```

## 📚 最佳实践

1. **使用语义化图标** - 选择最符合功能的图标
2. **保持一致尺寸** - 同一场景使用相同尺寸
3. **添加 aria-label** - 提升可访问性
4. **使用发光效果** - 增强赛博朋克风格
5. **动画适度使用** - 避免过度动画影响性能
