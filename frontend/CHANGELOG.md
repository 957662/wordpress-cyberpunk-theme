# 更新日志

## [0.2.0] - 2026-03-03

### 新增
- 添加增强版特效组件
  - CyberGrid - 赛博网格背景
  - DataStream - 数据流动画
  - HexagonGrid - 六边形网格
  - CircuitBoard - 电路板效果
- 新增 UI 组件
  - Badge - 徽章组件
  - Skeleton - 骨架屏组件
  - Tabs - 标签页组件
  - Accordion - 手风琴组件
  - Tooltip - 工具提示组件
- 新增工具函数
  - color-utils - 颜色处理工具
  - animation-utils - 动画工具函数
- 新增增强版展示页面
- 优化图标导出，添加统一的图标系统

### 改进
- 优化粒子背景性能
- 改进故障文字效果
- 更新项目文档
- 完善类型定义

### 修复
- 修复图标导出冲突问题
- 修复深色模式下的样式问题
