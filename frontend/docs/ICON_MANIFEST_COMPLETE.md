## 📐 图标设计规范

### 基础规范
```typescript
// 所有图标遵循统一的接口
interface BaseIconProps {
  size?: number;        // 尺寸：16, 20, 24, 32, 48
  className?: string;   // 自定义类名
  color?: string;       // 颜色（可选）
  variant?: 'solid' | 'outline'; // 实心/描边
}
```

### 尺寸规范
```css
/* 移动端 */
Mobile:  20px (导航), 16px (列表)

/* 桌面端 */
Desktop: 24px (标准), 32px (强调), 48px (展示)

/* 特殊用途 */
Favicon: 16px, 32px
Logo:    64px, 128px, 256px
```

### 颜色使用
```tsx
// 主要功能
<CyberIcon className="text-cyber-cyan" />

// 次要功能
<CyberIcon className="text-cyber-purple" />

// 警告提示
<WarningIcon className="text-cyber-yellow" />

// 错误状态
<ErrorIcon className="text-cyber-pink" />
```

### 特殊效果
```tsx
// 霓虹发光
<CyberIcon className="neon-glow-cyan" />

// 发光动画
<CyberIcon className="animate-pulse" />

// 旋转动画
<LoadingIcon className="animate-spin" />

// 故障效果
<GlitchIcon className="animate-glitch" />
```

## 🚀 高级用法

### 图标组合
```tsx
// 图标 + 文字
<div className="flex items-center gap-2">
  <DownloadIcon size={20} />
  <span>下载文件</span>
</div>

// 图标 + 计数
<div className="relative">
  <BellIcon size={24} />
  <span className="absolute -top-1 -right-1">
    5
  </span>
</div>
```

### 动态图标
```tsx
// 状态切换
const [isSaved, setIsSaved] = useState(false);

<BookmarkIcon
  size={24}
  variant={isSaved ? 'solid' : 'outline'}
  onClick={() => setIsSaved(!isSaved)}
/>

// 主题切换
const [isDark, setIsDark] = useState(true);

{isDark ? (
  <MoonIcon size={24} onClick={() => setIsDark(false)} />
) : (
  <SunIcon size={24} onClick={() => setIsDark(true)} />
)}
```

### 加载状态
```tsx
// 按钮加载
<Button disabled={loading}>
  {loading ? (
    <LoadingIcon size={20} className="animate-spin" />
  ) : (
    '提交'
  )}
</Button>

// 页面加载
<div className="flex justify-center">
  <LoaderIcon size={48} />
</div>
```
