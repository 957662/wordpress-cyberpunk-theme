## 💡 使用示例

### 基础用法

```tsx
import { SearchIcon, BlogIcon, UserIcon } from '@/components/icons';

// 默认使用
<SearchIcon />

// 自定义尺寸
<BlogIcon size={32} />

// 自定义颜色
<UserIcon size={24} variant="purple" />

// 自定义类名
<SearchIcon className="text-cyber-cyan hover:text-cyber-purple" />

// 启用动画
<LoadingIcon animated={true} />
```

### 高级用法

```tsx
// 带动画的加载图标
<LoadingIcon size={48} variant="cyan" animated={true} />

// 火箭图标
<RocketIcon size={64} variant="cyan" animated={true} />

// 奖杯图标
<TrophyIcon size={48} variant="yellow" animated={true} />

// 日夜切换
{isDark ? (
  <SunIcon size={24} variant="yellow" />
) : (
  <MoonIcon size={24} variant="purple" />
)}

// 心形图标
<HeartIcon
  size={24}
  variant={isLiked ? 'pink' : 'cyan'}
  className={isLiked ? 'fill-current' : ''}
/>
```

### 按钮图标

```tsx
<button className="flex items-center gap-2 px-4 py-2 bg-neon-gradient">
  <DownloadIcon size={20} variant="cyan" />
  <span>下载</span>
</button>
```

### 卡片标题图标

```tsx
<div className="flex items-center gap-3 mb-4">
  <BlogIcon size={32} variant="purple" />
  <h2 className="text-2xl font-bold">最新文章</h2>
</div>
```

## 🔧 Props 接口

### 基础图标属性

```typescript
interface BaseIconProps {
  size?: number;              // 图标尺寸（默认 24）
  variant?: 'cyan' | 'purple' | 'pink' | 'yellow' | 'green';
  className?: string;         // 自定义 Tailwind 类名
  animated?: boolean;         // 是否启用动画
}
```

### 特殊图标属性

```typescript
// ArrowIcon 额外属性
interface ArrowIconProps extends BaseIconProps {
  direction?: 'up' | 'down' | 'left' | 'right';
}

// SortIcon 额外属性
interface SortIconProps extends BaseIconProps {
  direction?: 'asc' | 'desc';
}

// ThemeIcon 额外属性
interface ThemeIconProps extends BaseIconProps {
  mode?: 'light' | 'dark';
}

// HeartIcon 额外属性
interface HeartIconProps extends BaseIconProps {
  filled?: boolean;
}
```

## 🎯 最佳实践

### ✅ 推荐

1. **使用标准尺寸**：优先使用 16, 20, 24, 32, 48px
2. **语义化颜色**：根据图标功能选择合适的颜色
3. **添加可访问性**：为图标添加 `aria-label`
4. **按需导入**：使用 tree-shaking 优化打包

```tsx
// 推荐：语义化颜色
<CheckIcon variant="green" />     // 成功
<WarningIcon variant="yellow" />  // 警告
<ErrorIcon variant="pink" />      // 错误

// 推荐：可访问性
<SearchIcon aria-label="搜索" />

// 推荐：按需导入
import { SearchIcon } from '@/components/icons';
```

### ❌ 避免

```tsx
// 避免：不规范的尺寸
<SearchIcon size={27} />

// 避免：颜色与语义不符
<ErrorIcon variant="green" />

// 避免：过度使用动画
<div>
  <LoadingIcon animated={true} />
  <RocketIcon animated={true} />
  <SparkleIcon animated={true} />
</div>
```

## 📊 性能优化

### Tree Shaking

```tsx
// ✅ 推荐：按需导入
import { SearchIcon } from '@/components/icons';

// ❌ 避免：全量导入
import * as Icons from '@/components/icons';
```

### CSS 控制

```tsx
// ✅ 推荐：使用 className
<SearchIcon className="text-cyber-cyan" />

// ❌ 避免：使用 inline style
<SearchIcon style={{ color: '#00f0ff' }} />
```
