## 📐 尺寸规格

标准尺寸选项：

```tsx
size={16}   // 超小：按钮内图标、列表
size={20}   // 小：输入框图标、小按钮
size={24}   // 标准：导航、卡片（默认）
size={32}   // 大：标题图标、功能入口
size={40}   // 超大：装饰图标、状态显示
size={48}   // 特大：英雄区图标、品牌展示
size={64}   // 巨大：背景装饰、特殊效果
```

## 🚀 性能优化

### Tree Shaking
```tsx
// ✅ 推荐：按需导入
import { SearchIcon } from '@/components/icons';

// ❌ 避免：全量导入
import * as Icons from '@/components/icons';
```

### 懒加载
```tsx
// 懒加载大型图标组件
const HeavyIcon = dynamic(() => import('@/components/icons/ComplexIcon'), {
  loading: () => <LoadingIcon />
});
```

### 代码分割
```tsx
// 按类别分组导入
import { HomeIcon, BlogIcon } from '@/components/icons/navigation';
import { GitHubIcon, TwitterIcon } from '@/components/icons/social';
```

## 📋 最佳实践

### ✅ DO - 推荐做法

1. **统一尺寸**: 同一区域使用相同尺寸
2. **语义化**: 根据含义选择合适的图标
3. **可访问性**: 添加 `aria-label`
4. **颜色一致**: 相同功能使用相同颜色
5. **按需导入**: 只导入需要的图标

### ❌ DON'T - 避免做法

1. **过度使用**: 不要堆砌图标
2. **随意尺寸**: 避免使用非标准尺寸
3. **颜色混乱**: 不要随意更改颜色
4. **全量导入**: 避免导入整个图标库
5. **忽略动画**: 合理使用动画增强体验
