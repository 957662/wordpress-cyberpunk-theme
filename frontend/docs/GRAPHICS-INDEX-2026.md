## 🎨 颜色使用

### 主要颜色
- **赛博青** `#00f0ff` - 用于主要强调、链接、按钮
- **赛博紫** `#9d00ff` - 用于渐变、装饰、次强调
- **激光粉** `#ff0080` - 用于警告、特殊状态
- **电压黄** `#f0ff00` - 用于高亮、通知

### 图标颜色
- 使用 `currentColor` 继承父元素颜色
- 内置 SVG 滤镜实现发光效果
- 支持 CSS 类动态变色

## 🚀 使用示例

### Logo 组件
```tsx
import { Logo } from '@/components/icons';

<Logo size={200} variant="main" className="hover:scale-110" />
```

### 图标使用
```tsx
import {
  NotificationIcon,
  ThemeToggleIcon,
  BrainAIIcon
} from '@/components/icons';

<NotificationIcon size={24} className="text-cyber-cyan" />
<ThemeToggleIcon size={20} className="text-cyber-purple" />
<BrainAIIcon size={32} className="text-cyber-pink" />
```

### 背景图案
```css
.hero-section {
  background-image: url('/backgrounds/circuit-pattern.svg');
  background-size: cover;
}
```

### 装饰元素
```tsx
<Image
  src="/decorations/tech-corner.svg"
  alt="Corner"
  width={100}
  height={100}
  className="absolute top-0 left-0"
/>
```
