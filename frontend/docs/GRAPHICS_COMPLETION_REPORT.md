## 🚀 使用示例

### 基础图标使用

```tsx
import { SearchIcon, HeartIcon, StarIcon } from '@/components/icons';

// 标准使用
<SearchIcon size={24} className="text-cyber-cyan" />

// 带颜色变体
<StarIcon size={24} filled variant="yellow" />

// 条件渲染
<HeartIcon size={24} filled={isLiked} variant="pink" />
```

### 装饰组件使用

```tsx
import {
  DataStreamDecoration,
  CircuitNodeDecoration,
  ScanlineEffect
} from '@/components/icons/CyberDecorations';

// 数据流装饰
<DataStreamDecoration color="#00f0ff" />

// 电路节点装饰
<CircuitNodeDecoration size={40} color="#9d00ff" />

// 扫描线效果
<div className="relative">
  <ScanlineEffect className="absolute inset-0" />
  <YourContent />
</div>
```

### 动画效果

```tsx
import { motion } from 'framer-motion';
import { EnergyRingDecoration } from '@/components/icons/CyberDecorations';

<motion.div
  animate={{ rotate: 360 }}
  transition={{ duration: 10, repeat: Infinity, ease: "linear" }}
>
  <EnergyRingDecoration size={60} color="#00f0ff" />
</motion.div>
```

## ✨ 核心特性

### 1. 霓虹发光效果

所有图标都内置 SVG 滤镜，实现赛博朋克风格的发光效果：

```tsx
<filter id="neonGlow" x="-50%" y="-50%" width="200%" height="200%">
  <feGaussianBlur in="SourceGraphic" stdDeviation="1" />
  <feMerge>
    <feMergeNode in="blur" />
    <feMergeNode in="SourceGraphic" />
  </feMerge>
</filter>
```

### 2. 颜色变体支持

部分图标支持多种颜色变体：

```tsx
<HeartIcon variant="pink" />    // 粉色
<HeartIcon variant="cyan" />    // 青色
<HeartIcon variant="purple" />  // 紫色
<HeartIcon variant="yellow" />  // 黄色
```

### 3. 填充模式

支持填充和轮廓两种模式：

```tsx
<StarIcon filled={false} />  // 轮廓
<StarIcon filled={true} />   // 填充
```

### 4. 动画集成

与 Framer Motion 无缝集成：

```tsx
<motion.div whileHover={{ scale: 1.1 }}>
  <SearchIcon size={24} />
</motion.div>
```
