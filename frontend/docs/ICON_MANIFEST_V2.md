## 🎨 使用方式

### 在 React 组件中使用

```tsx
// 1. 导入图标
import {
  SearchIcon,
  BlogIcon,
  GitHubIcon,
  CpuIcon,
  FileIcon,
  OnlineIcon,
  DiscordIcon
} from '@/components/icons';

// 2. 使用图标 (基础用法)
<SearchIcon size={24} />
<BlogIcon size={32} />

// 3. 使用赛博朋克变体
<CpuIcon size={48} variant="cyan" />
<CpuIcon size={64} variant="purple" />
<CpuIcon size={64} variant="pink" animated={true} />

// 4. 自定义样式
<SearchIcon className="text-cyber-cyan" size={20} />

// 5. 带动画效果
<SyncIcon size={24} variant="cyan" animated={true} />
<LoadingIcon size={32} variant="purple" />
```

### 可用变体 (Variants)

所有图标支持以下赛博朋克配色：

```tsx
variant="cyan"    // 霓虹青 #00f0ff (默认)
variant="purple"  // 赛博紫 #9d00ff
variant="pink"    // 激光粉 #ff0080
variant="yellow"  // 电压黄 #f0ff00
variant="green"   // 矩阵绿 #00ff88 (部分图标)
```

### 支持的属性

```tsx
interface IconProps {
  size?: number;        // 图标尺寸 (默认: 24)
  variant?: ColorVariant; // 颜色变体
  className?: string;    // 自定义类名
  animated?: boolean;    // 是否启用动画
}
```

## 📝 注意事项

1. 所有图标支持 TypeScript 类型检查
2. 使用 `currentColor` 实现动态颜色
3. 内置赛博朋克发光滤镜效果
4. 支持深色/浅色主题
5. SVG 格式确保任意分辨率清晰
6. 动画效果使用 Tailwind 动画类
7. 建议优先使用组件图标而非静态 SVG
