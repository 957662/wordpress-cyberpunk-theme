## 🎯 使用示例

### 使用数据流插画
```tsx
import { DataStream, NetworkFlow } from '@/components/graphics';

<DataStream width={400} height={200} animated={true} />
<NetworkFlow variant="animated" />
```

### 使用波浪组件
```tsx
import { CyberWave, WaveDivider } from '@/components/graphics';

<CyberWave variant="top" color="cyan" />
<WaveDivider color="purple" style="double" />
```

### 使用 SVG 背景
```tsx
<Image
  src="/backgrounds/data-flow-bg.svg"
  alt="Data Flow Background"
  fill
  className="object-cover"
/>
```
