### Logo 特性

- **渐变色系**: 霓虹青 (#00f0ff) → 赛博紫 (#9d00ff) → 激光粉 (#ff0080)
- **发光效果**: 内置 SVG 滤镜，支持霓虹发光
- **响应式**: 支持任意尺寸缩放
- **主题适配**: 深色/浅色主题兼容

### 渐变组合

#### 品牌渐变

```css
/* 主品牌渐变 */
gradient-brand: linear-gradient(135deg, #00f0ff 0%, #9d00ff 100%);

/* 三色渐变 */
gradient-tricolor: linear-gradient(90deg, #00f0ff 0%, #9d00ff 50%, #ff0080 100%);

/* 垂直渐变 */
gradient-vertical: linear-gradient(180deg, #0a0a0f 0%, #16162a 100%);
```

#### 功能渐变

```css
/* 成功渐变 */
gradient-success: linear-gradient(135deg, #00ff88 0%, #00f0ff 100%);

/* 警告渐变 */
gradient-warning: linear-gradient(135deg, #f0ff00 0%, #ff6600 100%);

/* 错误渐变 */
gradient-error: linear-gradient(135deg, #ff0080 0%, #ff6600 100%);
```

### 发光效果

## 📐 设计规范

### 尺寸规格

#### 图标

- **默认**: 24x24px
- **最小**: 16x16px
- **最大**: 64x64px
- **推荐**: 16, 20, 24, 32, 40, 48px

#### Logo

- **主 Logo**: 200x60px
- **方形**: 200x200px
- **图标**: 48x48px
- **Favicon**: 32x32px

#### 插画

- **小**: 200x200px
- **中**: 400x300px
- **大**: 800x600px

### 视觉特征

- **线条宽度**: 2px (标准), 1-2px (细节)
- **发光效果**: 所有元素带霓虹发光
- **科技元素**: 电路节点、连接线、角标装饰
- **渐变应用**: 多色渐变增强科技感

### 动画支持

- **旋转**: `LoadingIcon`
- **切换**: `ThemeIcon`
- **流动**: `data-stream.svg`
- **脉冲**: 部分发光效果

## 🎯 使用指南

### 在 React 组件中使用

```tsx
import Image from 'next/image';

// 使用 Logo
<Image src="/logo.svg" alt="CyberPress" width={200} height={60} />

// 使用图标
<Image src="/icons/github.svg" alt="GitHub" width={24} height={24} />

// 使用插画
<Image src="/illustrations/cyber-loading.svg" alt="Loading" width={400} height={300} />
```

### 直接在 HTML 中使用

```html
<!-- Logo -->
<img src="/logo.svg" alt="CyberPress" width="200" height="60">

<!-- 图标 -->
<img src="/icons/search.svg" alt="Search" width="24" height="24">

<!-- 插画 -->
<img src="/illustrations/empty-state.svg" alt="No results" width="400" height="300">
```

### CSS 背景使用

```css
.hero-section {
  background-image: url('/backgrounds/hero-bg.svg');
  background-size: cover;
  background-position: center;
}

.card {
  background-image: url('/patterns/grid.svg');
  background-size: 20px 20px;
}
```
