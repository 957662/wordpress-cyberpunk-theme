## 🎨 设计系统

### 配色方案
```css
霓虹青: #00f0ff  (主要色)
赛博紫: #9d00ff  (次要色)
激光粉: #ff0080  (强调色)
电压黄: #f0ff00  (警告色)
矩阵绿: #00ff88  (成功色)
```

### 标准尺寸
```tsx
16px  - 小尺寸
20px  - 中小尺寸
24px  - 标准尺寸 (默认)
32px  - 大尺寸
48px  - 超大尺寸
64px  - 特大尺寸
```

### 动画效果
- `animate-pulse` - 脉冲动画
- `animate-spin-slow` - 慢速旋转
- `animate-pulse` - 发光闪烁
