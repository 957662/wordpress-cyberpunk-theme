## 🚀 使用方式

### 导入图标组件
```tsx
// 单个导入
import { SearchIcon } from '@/components/icons';

// 批量导入
import * as Icons from '@/components/icons';

// 使用
<SearchIcon size={24} className="text-cyber-cyan" />
```

### 使用 SVG 文件
```tsx
import Image from 'next/image';

<Image
  src="/icons/cyber-logo-icon.svg"
  alt="CyberPress Logo"
  width={100}
  height={100}
/>
```

### 使用配色方案
```css
/* Tailwind CSS */
<button className="bg-cyber-cyan text-cyber-black">
  按钮
</button>

/* 自定义 CSS */
.my-element {
  color: #00f0ff;
  text-shadow: 0 0 10px #00f0ff;
}
```

## 🎓 设计建议

### 1. 色彩使用
- 主要操作使用霓虹青 (#00f0ff)
- 次要操作使用赛博紫 (#9d00ff)
- 危险操作使用激光粉 (#ff0080)
- 成功状态使用赛博绿 (#00ff88)

### 2. 图标选择
- 导航使用简洁的线性图标
- 功能使用直观的符号图标
- 装饰使用赛博朋克风格图标
- 状态使用带有颜色编码的图标

### 3. 动画应用
- 加载状态使用旋转动画
- 强调元素使用脉冲动画
- 数据传输使用流动动画
- 特殊效果使用故障动画

### 4. 响应式设计
- 移动端使用 16-20px 图标
- 桌面端使用 24px 图标
- 展示使用 32-48px 图标
- Logo 使用 64-256px
