## 💻 技术规格

### 图标组件
```typescript
interface BaseIconProps {
  size?: number;              // 图标尺寸（默认 24）
  variant?: 'cyan' | 'purple' | 'pink' | 'yellow' | 'green';
  className?: string;         // 自定义 Tailwind 类名
  animated?: boolean;         // 是否启用动画
}
```

### SVG 文件
- **版本**: SVG 1.1
- **压缩**: 优化过的路径数据
- **滤镜**: Gaussian Blur 发光效果
- **渐变**: LinearGradient 定义
- **动画**: CSS Animations

## 🎯 使用示例

### 图标使用
```tsx
import { SearchIcon, RocketIcon } from '@/components/icons';

// 基础使用
<SearchIcon size={24} variant="cyan" />

// 动画图标
<RocketIcon size={64} variant="cyan" animated={true} />
```

### SVG 使用
```tsx
import CyberCorner from '@/decorations/cyber-corner.svg';

<CyberCorner className="absolute top-0 left-0" />
```
