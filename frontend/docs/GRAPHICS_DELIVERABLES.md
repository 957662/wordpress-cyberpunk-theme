## ✨ 技术规格

### 图标组件
- **框架**: React 18+
- **语言**: TypeScript
- **格式**: SVG
- **样式**: Tailwind CSS 兼容
- **动画**: CSS Animations + SVG Filters

### SVG 文件
- **版本**: SVG 1.1
- **压缩**: 优化过的路径数据
- **滤镜**: 发光效果 (Gaussian Blur)
- **渐变**: LinearGradient 定义

### 性能优化
- **Tree Shaking**: 支持按需导入
- **内联 SVG**: 无额外 HTTP 请求
- **缓存友好**: 静态 SVG 文件
- **SSR 兼容**: 服务端渲染支持

## 🎯 使用建议

### 1. 图标使用
```tsx
// 按需导入
import { SearchIcon, BlogIcon } from '@/components/icons';

// 使用示例
<SearchIcon size={24} variant="cyan" />
<BlogIcon size={32} variant="purple" animated={true} />
```

### 2. 装饰元素使用
```tsx
// 作为背景
<Image src="/decorations/cyber-corner.svg" alt="装饰" />

// 作为组件
import CyberCorner from '@/decorations/cyber-corner.svg';
<CyberCorner className="absolute top-0 left-0" />
```

### 3. Logo 使用
```tsx
// 主 Logo
<Image src="/logos/cyberpress-icon.svg" alt="CyberPress" width={200} height={200} />

// Favicon
<link rel="icon" href="/logos/cyberpress-favicon.svg" />
```
