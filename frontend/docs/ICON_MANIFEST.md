## 📐 使用示例

### 基础使用
```tsx
import Image from 'next/image';

// 使用 SVG 图标
<Image
  src="/icons/search.svg"
  alt="Search"
  width={24}
  height={24}
  className="text-cyber-cyan"
/>

// 赛博朋克主题图标
<Image
  src="/icons/cyberpunk/microchip.svg"
  alt="Microchip"
  width={64}
  height={64}
/>
```

### 作为背景图案
```css
.hero-section {
  background-image: url('/patterns/cyber-grid.svg');
  background-repeat: repeat;
  background-size: 200px 200px;
}
```

### 作为插图使用
```tsx
<div className="relative h-96 w-full">
  <Image
    src="/illustrations/cyber-cityscape.svg"
    alt="Cyber City"
    fill
    className="object-cover"
  />
</div>
```

## 🎨 设计规范

### 尺寸规格
- **特小**: 16x16px (内联图标)
- **小**: 20x20px (紧凑布局)
- **中**: 24x24px (标准尺寸)
- **大**: 32x32px (功能图标)
- **特大**: 48x48px (展示图标)
- **Hero**: 64x64px+ (特殊展示)

### 视觉特征
- **线条宽度**: 1-2px（标准图标），2-3px（插画）
- **发光效果**: 使用 SVG `<filter>` 元素创建霓虹发光
- **科技元素**: 电路节点、连接线、角标装饰
- **渐变应用**: 线性渐变和径向渐变增强科技感
- **动画支持**: SVG `<animate>` 和 `<animateTransform>` 元素

### 动画类型
- **淡入淡出**: opacity 变化
- **缩放**: scale 变换
- **旋转**: rotate 变换
- **路径运动**: animateMotion
- **脉冲**: 呼吸效果
