## 🎯 使用示例

### 装饰元素使用

```tsx
import Image from 'next/image';

// 角落装饰
<Image src="/decorations/corner-accent.svg" alt="" width={100} height={100} className="absolute top-0 left-0" />

// 科技边框
<Image src="/decorations/tech-border.svg" alt="" width={400} height={20} className="w-full" />

// 分区分隔线
<Image src="/decorations/section-divider.svg" alt="" width={400} height={60} className="w-full" />
```

### 插画使用

```tsx
// 空状态插画
<Image src="/illustrations/cyberpunk-empty-state.svg" alt="No data" width={400} height={300} />

// 成功插画
<Image src="/illustrations/cyber-success.svg" alt="Success" width={400} height={300} />
```

### 赛博朋克图标使用

```tsx
// 能量场图标（带动画）
<Image src="/icons/cyberpunk/energy-field.svg" alt="Energy" width={48} height={48} />

// 等离子核心图标（带动画）
<Image src="/icons/cyberpunk/plasma-core.svg" alt="Plasma" width={48} height={48} />
```

## 📐 尺寸规格

标准尺寸选项：

```tsx
size={16}   // 超小：按钮内图标、列表
size={20}   // 小：输入框图标、小按钮
size={24}   // 标准：导航、卡片（默认）
size={32}   // 大：标题图标、功能入口
size={40}   // 超大：装饰图标、状态显示
size={48}   // 特大：英雄区图标、品牌展示
size={64}   // 巨大：背景装饰、特殊效果
```
