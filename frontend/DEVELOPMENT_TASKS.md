## 📝 开发规范

### 代码风格
- 使用 TypeScript 进行类型检查
- 遵循 ESLint 规则
- 使用 Prettier 格式化代码
- 遵循 React Hooks 规则

### 命名规范
- 组件使用 PascalCase
- 函数使用 camelCase
- 常量使用 UPPER_SNAKE_CASE
- 类型使用 PascalCase

### 文件组织
- 按功能模块组织文件
- 相关文件放在同一目录
- 使用 index.ts 导出公共接口
- 保持文件结构清晰
