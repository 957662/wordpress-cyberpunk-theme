## 📝 使用说明

### 基础用法
```tsx
<Image src="/icons/fileName.svg" alt="Description" width={24} height={24} />
```

### 带样式
```tsx
<Image
  src="/icons/tech-hexagon.svg"
  alt="Tech Hexagon"
  width={48}
  height={48}
  className="hover:scale-110 transition-transform"
/>
```

### CSS 背景
```css
background-image: url('/patterns/hex-grid.svg');
```
