## 🔧 技术规格

### 图标属性
- **格式**: SVG
- **描边宽度**: 2px
- **描边样式**: round caps & joins
- **颜色**: `currentColor` (继承父元素颜色)
- **尺寸**: 24x24 (标准), 可缩放
- **发光效果**: 内置霓虹滤镜

### 发光滤镜
所有图标都包含自定义SVG滤镜：
```xml
<filter id="neon-glow-*">
  <feGaussianBlur stdDeviation="2" result="blur"/>
  <feMerge>
    <feMergeNode in="blur"/>
    <feMergeNode in="SourceGraphic"/>
  </feMerge>
</filter>
```

## 🎨 使用方式

### 1. 直接使用 SVG 文件

```tsx
import Image from 'next/image';

<Image src="/icons/cpu.svg" alt="CPU" width={24} height={24} />
```

### 2. 使用 Lucide React（推荐）

项目已安装 `lucide-react`：

```tsx
import {
  Cpu,
  Trophy,
  Target,
  BarChart3,
  TrendingUp,
  Activity,
  Zap,
  Rocket,
  Bot,
  Fingerprint,
  GitMerge,
  Bell,
  Link2,
  FileText,
  ImagePlus,
  VideoOff,
} from 'lucide-react';

<Cpu className="w-6 h-6 text-cyber-cyan" />
<Trophy className="w-8 h-8 text-cyber-yellow" />
```

### 3. 自定义组件

```tsx
// components/Icon.tsx
export function Icon({ name, size = 24, className = '' }) {
  return (
    <Image
      src={`/icons/${name}.svg`}
      alt={name}
      width={size}
      height={size}
      className={className}
    />
  );
}

// 使用
<Icon name="cpu" size={32} className="text-cyber-cyan" />
```
