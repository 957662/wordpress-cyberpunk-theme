# CyberPress 图形设计项目总结

## 🎯 项目概述

CyberPress 是一个采用赛博朋克设计风格的技术平台，具有完整的图形设计系统。本项目提供了一套完整的 Logo、图标、插画、装饰元素和品牌素材。

## 🎨 设计系统

### 配色方案

```css
/* 主色 */
--cyber-cyan:      #00f0ff  /* 霓虹青 - 主要色 */
--cyber-purple:    #9d00ff  /* 赛博紫 - 次要色 */
--cyber-pink:      #ff0080  /* 激光粉 - 强调色 */

/* 辅助色 */
--cyber-green:     #00ff88  /* 矩阵绿 - 成功色 */
--cyber-yellow:    #f0ff00  /* 电压黄 - 警告色 */
--cyber-orange:    #ff6600  /* 火焰橙 - 提示色 */

/* 背景色 */
--cyber-dark:      #0a0a0f  /* 主背景 */
--cyber-darker:    #050508  /* 次背景 */
--cyber-card:      #16162a  /* 卡片背景 */
--cyber-border:    #2a2a4a  /* 边框颜色 */
```

### 视觉特征

- ✅ **赛博朋克风格**: 霓虹发光、科技线条、未来感
- ✅ **渐变应用**: 多色渐变增强视觉层次
- ✅ **电路元素**: 节点、连接线装饰
- ✅ **一致性强**: 统一的设计语言
- ✅ **发光效果**: 内置 SVG 滤镜实现霓虹效果

### SVG 特性

- ✅ **矢量格式**: 可任意缩放不失真
- ✅ **内置滤镜**: 霓虹发光效果
- ✅ **渐变填充**: 丰富的色彩层次
- ✅ **轻量级**: 文件体积小，加载快
- ✅ **可定制**: 易于修改颜色和样式
- ✅ **动画支持**: 部分元素支持动画效果

## 🔧 使用指南

### 图标使用

#### 直接使用 SVG
```tsx
<Image src="/icons/home.svg" alt="Home" width={24} height={24} />
```

#### 作为组件使用
```tsx
import { HomeIcon } from '@/components/icons';

<HomeIcon size={24} className="text-cyber-cyan" />
```

#### 使用新增赛博图标
```tsx
<Image src="/icons/neon-wifi.svg" alt="WiFi" width={24} height={24} />
<Image src="/icons/data-pulse.svg" alt="Data" width={24} height={24} />
```

### 插画使用

```tsx
// AI 助手插画
<Image src="/illustrations/ai-assistant.svg" alt="AI" width={400} height={400} />

// DevOps 流水线
<Image src="/illustrations/cyber-devops.svg" alt="DevOps" width={400} height={400} />

// 安全扫描
<Image src="/illustrations/security-scan.svg" alt="Security" width={400} height={400} />
```

### 装饰元素使用

```tsx
// 六边形加载器
<Image src="/decorations/hex-loader.svg" alt="Loading" className="animate-spin" />

// 赛博按钮
<div className="relative">
  <Image src="/decorations/cyber-button.svg" alt="" className="absolute" />
  <button>Click Me</button>
</div>

// 动画电路
<Image src="/decorations/circuit-lines-animated.svg" alt="Circuit" />

// 数据流
<Image src="/decorations/data-flow.svg" alt="Data Flow" />
```

### 品牌素材使用

```tsx
// 单色版本
<Image src="/brand/logo-monochrome.svg" alt="Logo" />

// 线框版本
<Image src="/brand/logo-outline.svg" alt="Logo" />
```
