## 💻 Usage Examples

### Next.js Image Component
```tsx
import Image from 'next/image'

<Image
  src="/icons/home.svg"
  alt="Home"
  width={24}
  height={24}
  className="text-cyber-cyan"
/>
```

### React Components
```tsx
import { HomeIcon, BlogIcon, SearchIcon } from '@/components/icons'

<HomeIcon className="text-cyber-cyan hover:shadow-neon-cyan" size={24} />
```

### Direct SVG Import
```tsx
import HomeSvg from '/icons/home.svg'

<HomeSvg className="w-6 h-6 text-cyber-cyan" />
```

### Inline SVG
```tsx
<svg className="w-6 h-6">
  <use href="/icons/sprite.svg#home" />
</svg>
```
