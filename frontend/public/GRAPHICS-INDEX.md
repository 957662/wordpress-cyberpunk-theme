## 🎨 使用指南

### 快速引用
```tsx
// 图标
<Image src="/icons/cyber-github.svg" width={24} height={24} alt="GitHub" />

// Logo
<Image src="/assets/logo/cyberpress-logo.svg" width={200} height={60} alt="Logo" />

// 背景
<div style={{ backgroundImage: 'url(/patterns/cyber-grid.svg)' }} />

// 装饰
<Image src="/decorations/cyber-corner.svg" width={100} height={100} alt="" />

// 插画
<Image src="/illustrations/cyber-server.svg" width={400} height={300} alt="Server" />
```

### CSS 背景使用
```css
.hero-section {
  background-image: url('/patterns/cyber-grid.svg');
  background-size: cover;
  background-position: center;
}

.card {
  background-image: url('/backgrounds/card-bg.svg');
  background-size: 100% 100%;
}
```

## 📝 自定义开发

### 创建新图标
1. 复制现有图标作为模板

3. 添加渐变和动画
4. 优化文件大小
5. 更新文档

1. 定义 `<pattern>` 元素
2. 添加重复单元
3. 设置适当间距
4. 添加透明度和渐变
5. 测试平铺效果

## 🎯 设计原则

1. **一致性** - 保持统一的视觉语言
2. **可读性** - 确保图标清晰可辨
3. **性能** - 优化 SVG 代码大小
4. **可访问性** - 添加适当的 alt 文本
5. **动画** - 使用流畅的过渡效果
