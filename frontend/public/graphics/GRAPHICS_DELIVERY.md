## 🎯 设计规范

### 视觉特征

- **配色方案**: 霓虹青 (#00f0ff) → 赛博紫 (#9d00ff) → 激光粉 (#ff0080)
- **发光效果**: SVG 滤镜实现霓虹发光
- **科技元素**: 电路节点、连接线、角标装饰
- **渐变应用**: 多色渐变增强科技感

### 尺寸规格

- **图标**: 默认 24x24px，支持 16-64px
- **Logo**: 32x32px 到 200x200px
- **插画**: 200x200px 到 400x300px
- **装饰**: 100x100px 到 400x60px

### 格式

- **矢量格式**: SVG
- **响应式**: 支持任意尺寸缩放
- **优化**: 内联 SVG，无额外请求

## 📝 使用说明

### 在 React 组件中使用

```tsx
import Image from 'next/image';

// Logo
<Image src="/logo.svg" alt="CyberPress" width={200} height={60} />

// 图标
<Image src="/icons/github.svg" alt="GitHub" width={24} height={24} />

// 插画
<Image src="/illustrations/cyber-loading.svg" alt="Loading" />
```

### CSS 背景使用

```css
.hero-section {
  background-image: url('/backgrounds/hero-bg.svg');
}

.card {
  background-image: url('/patterns/grid.svg');
}
```
