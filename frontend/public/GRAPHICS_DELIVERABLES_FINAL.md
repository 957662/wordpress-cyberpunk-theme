## 🎨 设计规范

### 视觉特征
- ✅ **赛博朋克风格**: 霓虹发光、科技线条
- ✅ **渐变应用**: 多色渐变增强层次
- ✅ **电路元素**: 节点、连接线装饰
- ✅ **一致性强**: 统一的设计语言

### 配色系统
```css
/* 主色 */
--cyber-cyan:      #00f0ff  /* 霓虹青 */
--cyber-purple:    #9d00ff  /* 赛博紫 */

/* 辅助色 */
--cyber-green:     #00ff88  /* 矩阵绿 */
--cyber-yellow:    #f0ff00  /* 电压黄 */

/* 背景色 */
--cyber-dark:      #0a0a0f  /* 主背景 */
--cyber-darker:    #050508  /* 次背景 */
--cyber-card:      #16162a  /* 卡片背景 */
```

### SVG 特性
- ✅ **矢量格式**: 可任意缩放不失真
- ✅ **内置滤镜**: 霓虹发光效果
- ✅ **渐变填充**: 丰富的色彩层次
- ✅ **轻量级**: 文件体积小，加载快
- ✅ **可定制**: 易于修改颜色和样式

## 🔧 使用指南

### 图标使用

#### 方式一: 直接使用 SVG
```tsx
<Image src="/icons/home.svg" alt="Home" width={24} height={24} />
```

#### 方式二: 作为 React 组件
```tsx
import { HomeIcon } from '@/components/icons';

<HomeIcon size={24} className="text-cyber-cyan" />
```

#### 方式三: 使用新增赛博图标
```tsx
<Image src="/icons/neon-wifi.svg" alt="WiFi" width={24} height={24} />
<Image src="/icons/data-pulse.svg" alt="Data" width={24} height={24} />
```

### 插画使用

```tsx
// 页面状态
<Image src="/illustrations/ai-assistant.svg" alt="AI" width={400} height={400} />

// 英雄区域
<div className="hero">
  <Image src="/illustrations/cyber-connection.svg" alt="Network" />
</div>
```

### 装饰元素使用

```tsx
// 加载动画
<Image src="/decorations/hex-loader.svg" alt="Loading" className="animate-spin" />

// 按钮背景
<div className="relative">
  <Image src="/decorations/cyber-button.svg" alt="" className="absolute" />
  <button>Click Me</button>
</div>
```

### 品牌素材使用

```tsx
// 单色版本
<Image src="/brand/logo-monochrome.svg" alt="Logo" />

// 线框版本
<Image src="/brand/logo-outline.svg" alt="Logo" />
```
