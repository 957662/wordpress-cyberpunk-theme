## 🎨 配色方案

所有图标都使用赛博朋克配色系统：

### 主色
- 霓虹青: `#00f0ff`
- 赛博紫: `#9d00ff`
- 激光粉: `#ff0080`

### 辅助色
- 矩阵绿: `#00ff88`
- 电压黄: `#f0ff00`
- 火焰橙: `#ff6600`

### 发光效果
所有图标都内置 SVG 滤镜实现霓虹发光效果。

## 📦 使用方式

### 直接使用 SVG
```tsx
<Image src="/icons/home.svg" alt="Home" width={24} height={24} />
```

### 作为组件使用
```tsx
import { HomeIcon } from '@/components/icons';

<HomeIcon size={24} />
```

### 自定义样式
```tsx
<HomeIcon size={24} className="text-cyber-cyan" />
```
