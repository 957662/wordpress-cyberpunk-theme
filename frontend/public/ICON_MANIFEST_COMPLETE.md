## 🔄 Animated Icons

### Loading Animation
The `loading.svg` icon includes a built-in rotation animation:
```svg
<animateTransform
  attributeName="transform"
  type="rotate"
  from="0 12 12"
  to="360 12 12"
  dur="1s"
  repeatCount="indefinite"
/>
```

### Pulse Animation
For pulsing glow effects, add CSS:
```css
@keyframes neon-pulse {
  0%, 100% { opacity: 1; }
  50% { opacity: 0.8; }
}

.icon-pulse {
  animation: neon-pulse 2s ease-in-out infinite;
}
```

## 📝 Icon Naming Convention

- **kebab-case**: All files use kebab-case (e.g., `external-link.svg`)
- **Descriptive**: Names should clearly indicate the icon's purpose
- **Consistent**: Similar icons share prefixes (e.g., `chevron-*`)

---

## 🎯 Best Practices

1. **Size**: Use 24x24 viewBox for consistency
2. **Stroke**: 2px stroke width for optimal visibility
3. **Colors**: Apply theme colors via CSS, not hardcoded
4. **Accessibility**: Always include `alt` text or `aria-label`
5. **Performance**: Use Next.js Image component for automatic optimization
6. **Animation**: Keep animations subtle and purposeful
