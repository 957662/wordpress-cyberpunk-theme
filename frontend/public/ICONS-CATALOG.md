## 🎯 使用示例

### React 组件中使用
```tsx
import Image from 'next/image';

// 使用图标
<Image src="/icons/cyber-github.svg" alt="GitHub" width={24} height={24} />
<Image src="/icons/cyber-heart.svg" alt="Like" width={20} height={20} />
```

### 使用 Icon 组件
```tsx
import { Icon } from '@/components/graphics/Icon';

// 基础用法
<Icon name="cyber-github" size={24} variant="social" />
<Icon name="cyber-heart" size={20} glow={true} animation="pulse" />
```

## 🌈 颜色主题

### 霓虹青 (Cyan)
- **代码**: `#00f0ff`
- **用途**: 主要交互元素、链接、科技感

### 赛博紫 (Purple)
- **代码**: `#9d00ff`
- **用途**: 次要元素、装饰、渐变

### 激光粉 (Pink)
- **代码**: `#ff0080`
- **用途**: 强调、警告、情感按钮

### 电压黄 (Yellow)
- **代码**: `#f0ff00`
- **用途**: 高亮、提示、能量元素

### 深空黑 (Black)
- **代码**: `#0a0a0f`
- **用途**: 背景、基础色

## ✨ 动画效果

所有赛博朋克图标都支持以下动画效果：

| 动画 | 类名 | 效果 |
|------|------|------|
| 脉冲 | `animate-pulse` | 透明度变化 |
| 旋转 | `animate-spin` | 360度旋转 |
| 弹跳 | `animate-bounce` | 上下弹跳 |
| 发光 | `drop-shadow` | 霓虹发光效果 |

## 📐 图标规格

- **尺寸**: 24x24px (标准)
- **格式**: SVG
- **描边**: 2px (标准)
- **圆角**: stroke-linecap="round"
- **视图框**: viewBox="0 0 24 24"

---

## 🚀 自定义图标

创建新的赛博朋克图标时，请遵循以下规范：

1. **使用渐变色**: 从 `#00f0ff` 到 `#9d00ff` 或 `#ff0080`
2. **添加动画**: 使用 SVG `<animate>` 标签
3. **发光效果**: 添加 `<filter>` 元素
4. **保持一致**: 使用相同的描边宽度和样式
