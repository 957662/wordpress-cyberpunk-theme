## 🎯 使用指南

### 导入方式

#### 方式一：从统一索引导入（推荐）
```tsx
import {
  HomeIcon,
  ShieldIcon,
  MainLogo,
  CyberGridPattern,
  RobotIllustration,
  SocialIcons
} from '@/components/graphics';
```

#### 方式二：从具体文件导入
```tsx
import { HomeIcon } from '@/components/graphics/SVGIcons';
import { ShieldIcon } from '@/components/graphics/AdditionalIcons';
import { MainLogo } from '@/components/graphics/Logos';
```

### 基础使用

#### 图标
```tsx
<HomeIcon size={24} />
<ShieldIcon size={32} glow />
<RocketIcon size={48} color="#00f0ff" animated />
```

#### Logo
```tsx
<MainLogo width={200} />
<SquareLogo size={64} animated />
<AnimatedLogo size={100} />
```

#### 背景
```tsx
<CyberGridPattern opacity={0.1} />
<HexagonPattern color="#9d00ff" animated />
<MatrixPattern opacity={0.15} />
```

#### 插画
```tsx
<CyberCityIllustration width={600} animated />
<RobotIllustration width={400} />
<NeuralNetworkIllustration width={500} animated />
```

#### 组合
```tsx
<SocialIcons variant="circle" size={32} />
<TechStackIcons layout="grid" columns={4} />
<FeatureIcons animated />
```

## ⚡ 动画效果

所有组件都支持以下动画效果：

1. **SMIL 动画** - SVG 原生动画（自动播放）
2. **CSS 动画** - 通过类名触发
3. **Framer Motion** - React 集成动画

### 启用动画
```tsx
// 通过 props
<Component animated />

// 通过 className
<Component className="animate-spin" />
```
