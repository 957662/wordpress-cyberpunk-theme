📋 快速开始
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 查看集成指南
   $ cat INTEGRATION_GUIDE.md

2. 配置环境变量
   $ cp .env.example .env.production
   $ # 编辑 .env.production

3. 安装依赖
   $ cd backend && pip install -r requirements.txt
   $ cd ../frontend && npm install

4. 运行测试
   $ cd backend && pytest
   $ cd ../frontend && npm test

5. 启动服务
   $ docker-compose up -d

6. 验证功能
   $ curl http://localhost:8000/api/v1/wordpress/posts
