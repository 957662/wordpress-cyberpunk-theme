## 📝 使用方法

### 快速开始

#### 1. 安装依赖
```bash
npm install framer-motion date-fns lucide-react clsx tailwind-merge
```

#### 2. 导入组件
```tsx
import {
  ArticleSummaryAI,
  ArticleVoiceReader,
  ArticleVersionHistory,
  CollaborativeAnnotations
} from '@/components/blog/advanced';
```

#### 3. 使用组件
```tsx
<ArticleSummaryAI
  content={post.content}
  title={post.title}
/>

<ArticleVoiceReader
  content={post.content}
  autoHighlight
/>

<ArticleVersionHistory
  articleId={post.id}
  versions={post.versions}
/>

<CollaborativeAnnotations
  content={post.content}
  articleId={post.id}
  userId={user.id}
  currentUser={user}
/>
```
