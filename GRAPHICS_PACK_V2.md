## 🎯 使用指南

### 快速开始

#### 1. 使用 Logo
```tsx
import Image from 'next/image';

// 主 Logo
<Image src="/logo-main.svg" alt="CyberPress" width={200} height={200} />

// Logo 图标
<Image src="/logo-icon.svg" alt="CyberPress" width={100} height={100} />

// Favicon
<link rel="icon" href="/logo-favicon.svg" />
```

#### 2. 使用图标
```tsx
// 方式1: 使用 Next.js Image
<Image src="/icons/home.svg" alt="Home" width={24} height={24} />

// 方式2: 使用 lucide-react (推荐)
import { Home, Search, Github, Twitter } from 'lucide-react';
<Home className="w-6 h-6 text-cyber-cyan" />

// 方式3: 作为 SVG 组件
export function SearchIcon() {
  return (
    <svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 24 24" fill="none" stroke="currentColor">
      <circle cx="11" cy="11" r="8"/>
      <path d="m21 21-4.3-4.3"/>
    </svg>
  );
}
```

#### 3. 使用装饰元素
```tsx
// 角标装饰
<div className="relative p-6 cyber-card">
  <Image
    src="/decorations/corner-accent.svg"
    alt=""
    className="absolute top-0 left-0 w-16 h-16 pointer-events-none"
  />
  <h3>Card with cyber corners</h3>
</div>

// 分隔线
<Image
  src="/decorations/divider-cyber.svg"
  alt=""
  className="w-full h-5 my-8"
/>
```

#### 4. 使用插画
```tsx
// 404 页面
<div className="flex flex-col items-center">
  <Image
    src="/illustrations/404-glitch.svg"
    alt="404 - Page Not Found"
    width={400}
    height={300}
    priority
  />
  <h2 className="text-2xl font-bold text-cyber-cyan mt-6">404</h2>
  <p className="text-cyber-gray-200">Page not found</p>
</div>

// 空状态
<Image src="/illustrations/empty-state.svg" alt="No content" />
```

#### 5. 使用背景图案
```tsx
// CSS 方式
<div className="bg-grid-pattern">
  {/* 内容 */}
</div>

// 或直接使用 Image
<Image
  src="/patterns/grid.svg"
  alt=""
  fill
  className="object-cover opacity-10"
/>
```

## 🚀 集成到项目

### 1. 更新 app/layout.tsx
```tsx
// 添加 Favicon
export const metadata = {
  title: 'CyberPress Platform',
  description: 'A cyberpunk-styled blog platform',
  icons: {
    icon: '/logo-favicon.svg',
  },
};
```

### 2. 创建图标组件
```tsx
// components/Icon.tsx
export function Icon({ name, size = 24 }: { name: string; size?: number }) {
  return (
    <Image
      src={`/icons/${name}.svg`}
      alt={name}
      width={size}
      height={size}
    />
  );
}

// 使用
<Icon name="home" size={24} />
<Icon name="github" size={32} />
```

### 3. 创建 404 页面
```tsx
// app/not-found.tsx
export default function NotFound() {
  return (
    <div className="flex flex-col items-center justify-center min-h-screen">
      <Image
        src="/illustrations/404-glitch.svg"
        alt="404"
        width={400}
        height={300}
        priority
      />
      <h1 className="text-4xl font-bold text-cyber-cyan mt-8">404</h1>
      <p className="text-cyber-gray-200 mt-4">Page not found</p>
      <Link href="/" className="cyber-button mt-6">
        Go Home
      </Link>
    </div>
  );
}
```

### 4. 创建空状态组件

## 🔧 自定义和扩展

### 修改颜色
使用 CSS filter 改变图标颜色：
```css
.icon-pink {
  filter: hue-rotate(180deg);
}
```

### 调整大小
所有 SVG 都是可缩放的：
```tsx
<Image src="/icons/home.svg" width={16} height={16} />  {/* 小 */}
<Image src="/icons/home.svg" width={24} height={24} />  {/* 中 */}
<Image src="/icons/home.svg" width={32} height={32} />  {/* 大 */}
```

### 添加动画
```css
@keyframes pulse {
  0%, 100% { opacity: 1; }
  50% { opacity: 0.5; }
}

.icon-pulse {
  animation: pulse 2s ease-in-out infinite;
}
```
