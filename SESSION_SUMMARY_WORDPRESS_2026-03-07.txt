🔗 快速开始
================================================================================

1. 配置环境变量 (.env.local):
   NEXT_PUBLIC_WORDPRESS_URL=https://your-wordpress-site.com

2. 客户端使用:
   import { useWordPressPosts } from '@/lib/wordpress';
   
3. 服务端使用:
   import { getServerPosts } from '@/lib/wordpress/server-client';

4. 查看示例:
