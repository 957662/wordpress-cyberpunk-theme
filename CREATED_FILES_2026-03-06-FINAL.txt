## WordPress 集成层

### 1. API 客户端
- frontend/lib/wordpress/client-new.ts
  * WordPress API 客户端
  * 统一管理所有 WordPress API 请求
  * 支持文章、分类、标签、作者、评论等 CRUD 操作
  * 包含错误处理和超时机制

### 2. 数据适配器
- frontend/lib/wordpress/adapter.ts
  * WordPress 数据格式转换
  * 将 WordPress API 原始数据转换为应用格式
  * 包含文章、分类、标签、作者、评论的适配器
  * 提供映射表创建和辅助函数

### 3. React Query Hooks
- frontend/lib/wordpress/hooks-new.ts
  * 类型安全的数据获取 hooks
  * 包含 usePosts, usePost, useCategories, useTags 等
  * 支持缓存和自动重新验证
  * 包含变更 hooks (useCreatePost, useUpdatePost 等)

### 4. 集成导出
- frontend/lib/wordpress/index.ts
  * WordPress 模块统一导出
  * 简化导入路径
