## 🚀 使用方法

### 快速开始

1. **导入组件**

```typescript
import {
  ReadingProgress,
  FontSizeAdjuster,
  ShareButtons,
  BookmarkButton,
  PrintButton,
} from '@/components/blog';
```

2. **在页面中使用**

```tsx
export default function ArticlePage() {
  return (
    <article>
      <ReadingProgress showPercentage />
      
      <div className="max-w-4xl mx-auto px-4">
        <div className="flex justify-between mb-6">
          <FontSizeAdjuster />
          <div className="flex gap-2">
            <BookmarkButton postId="123" title="文章标题" />
            <PrintButton />
          </div>
        </div>

        <div data-article-content>
          {/* 文章内容 */}
        </div>

        <ShareButtons title="文章标题" />
      </div>
    </article>
  );
}
```

### 详细文档

查看 `NEW_USER_FEATURES_GUIDE.md` 获取完整的使用指南。
