## 📦 使用示例

```typescript
// 使用图表组件
import { BarChart } from '@/components/charts';

<BarChart 
  data={[
    { label: '一月', value: 100 },
    { label: '二月', value: 200 },
  ]}
  colorScheme="cyan"
/>

// 使用计时器
import { Stopwatch } from '@/components/timer';

<Stopwatch colorScheme="purple" />

// 使用网络状态
import { NetworkStatus } from '@/components/network';

<NetworkStatus showDetails position="top-right" />

// 使用上传组件
import { FileDropzone } from '@/components/upload';

<FileDropzone 
  onUpload={handleUpload}
  maxFiles={5}
  maxSize={10 * 1024 * 1024}
/>
```

## 🔧 配置说明

所有组件都支持以下配置：
- `colorScheme`: 颜色主题
- `className`: 自定义样式类名
- 响应式设计，支持移动端
