                        📝 使用说明
================================================================================

组件导入:
```typescript
import { GitHubProfile, SocialShare } from '@/components/integrations';
import { LazyImage } from '@/components/performance';
import { Tabs, QuickTabs } from '@/components/common';
import { apiClient, cache } from '@/services';
```

组件使用:
```typescript
// GitHub 集成
<GitHubProfile username="username" showStats showRepos />

// 社交分享
<SocialShare url="https://example.com" title="Check this out!" />

// 懒加载图片
<LazyImage src="/image.jpg" alt="Description" loading="lazy" />

// 标签页
<QuickTabs tabs={[...]} />

// API 调用
const data = await apiClient.get('/posts');

// 缓存操作
cache.set('key', data, 60000);
```
