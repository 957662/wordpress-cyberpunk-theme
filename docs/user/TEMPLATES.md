## 功能概述

**Portfolio** 是 Cyberpunk Theme 的核心功能，让你以专业的方式展示你的作品集。

### 主要特点

- ✨ **精美展示**: 赛博朋克风格的卡片布局
- 🏷️ **分类过滤**: 按类别筛选项目

- 🔗 **外链支持**: 链接到外部项目
- 🎯 **SEO 优化**: 搜索引擎友好

## 显示 Portfolio

### 方法 A: 使用 Portfolio 页面模板

![页面模板](../assets/images/portfolio-display-1.png)

1. **页面 > 新建页面**
2. 输入标题: "Portfolio" 或 "作品集"
3. 右侧 **页面属性** > **模板**
4. 选择 **Portfolio Archive**
5. 发布页面

**效果**: 显示所有 Portfolio 项目，支持分类过滤

---

### 方法 B: 使用短代码

在任何页面或文章中：

```shortcode
[portfolio count="6" columns="3" filter="yes"]
```

#### 短代码参数

| 参数 | 说明 | 默认值 | 可选值 |
|------|------|--------|--------|
| `count` | 显示项目数 | -1 (全部) | 任意数字 |
| `columns` | 列数 | 3 | 1, 2, 3, 4 |
| `filter` | 显示过滤 | yes | yes, no |
| `category` | 分类筛选 | all | 分类 slug |
| `orderby` | 排序方式 | date | date, title, rand |

#### 短代码示例

```shortcode
<!-- 显示 6 个项目，2 列，显示过滤 -->
[portfolio count="6" columns="2" filter="yes"]

<!-- 显示 Web Design 分类的所有项目 -->
[portfolio category="web-design"]

<!-- 随机显示 3 个项目 -->
[portfolio count="3" orderby="rand"]
```

## 📦 安装问题

### 主题激活后样式错乱？

**症状**:
- 网站没有样式
- 布局混乱
- 图片不显示

**解决方案**:

1. **清除缓存**
   ```
   浏览器缓存: Ctrl + Shift + Delete
   WordPress 缓存: 安装 WP Super Clear Cache 插件
   ```

2. **检查 PHP 版本**
   ```
   仪表盘 > 工具 > 站点健康
   要求: PHP 7.4 或更高
   ```

3. **禁用冲突插件**
   ```
   插件 > 已安装的插件
   批量操作 > 停用所有
   逐个重新启用，找出冲突插件
   ```

4. **重新安装主题**
   ```
   外观 > 主题 > 主题详情
   删除 > 重新上传
   ```

> 💡 **提示**: 90% 的样式问题可以通过清除缓存解决

## 🎨 定制问题

### 如何修改霓虹颜色？

**快速方法**: 使用主题定制器

1. **外观 > 自定义**
2. **颜色 > 主色调**
3. 选择颜色或输入十六进制代码
4. 实时预览效果
5. 点击 **发布**

**预设颜色方案**:
```
霓虹青:  #00f0ff (默认)
霓虹品红: #ff00ff
霓虹黄:  #f0ff00
霓虹绿:  #00ff00
霓虹紫:  #9d00ff
```

**自定义颜色**:
```
使用任何十六进制颜色代码
例如: #ff6b6b (珊瑚红)
```

### 如何更换字体？

**方法 A**: 使用主题定制器

```
外观 > 自定义 > 字体
```

**可用字体**:
- Courier New (默认，赛博朋克风格)
- Roboto
- Open Sans
- Lato
- Montserrat

**方法 B**: 使用 Google Fonts

1. 访问 [Google Fonts](https://fonts.google.com/)
2. 选择字体
3. 复制 embed 代码
4. 外观 > 自定义 > 自定义 CSS
5. 粘贴代码并应用

```css
/* 示例: 使用 Orbitron 字体 */
@import url('https://fonts.googleapis.com/css2?family=Orbitron:wght@400;700&display=swap');

body {
    font-family: 'Orbitron', monospace;
}
```

### 如何禁用扫描线效果？

**步骤**:

1. **外观 > 自定义**
2. **高级 > 视觉效果**
3. **扫描线效果** > **关闭**
4. **发布**

**或者使用 CSS**:

```css
/* 添加到外观 > 自定义 > 自定义 CSS */
body::before {
    display: none !important;
}
```

## 🔥 紧急问题

### 白屏错误 (White Screen of Death)

**症状**:
- 网站完全空白
- 无错误信息显示
- 前端或后台空白

**诊断方法**:

1. **启用调试模式**
   ```php
   // 编辑 wp-config.php
   define('WP_DEBUG', true);
   define('WP_DEBUG_DISPLAY', true);
   ```

2. **检查错误日志**
   ```
   /wp-content/debug.log
   ```

**解决方案**:

**方法 A: 增加内存限制**
```php
// 编辑 wp-config.php
define('WP_MEMORY_LIMIT', '256M');
```

**方法 B: 禁用所有插件**
```
1. 通过 FTP 或文件管理器
2. 重命名 /wp-content/plugins/ 文件夹
3. 检查网站是否恢复
```

**方法 C: 切换到默认主题**
```
1. 重命名主题文件夹
   /wp-content/themes/cyberpunk/ → /cyberpunk-old/
2. WordPress 自动切换到默认主题
3. 检查网站是否恢复
```

**方法 D: 检查 PHP 错误**
```php
// 常见错误:
Fatal error: Allowed memory size exhausted
→ 解决: 增加内存限制

Fatal error: Class 'xxx' not found
→ 解决: 重新安装主题

Fatal error: Call to undefined function
→ 解决: 更新 PHP 版本
```

**方法 B: 创建新管理员**

// 添加到 functions.php
function create_admin_user() {
    $username = 'new_admin';
    $password = 'secure_password';
    $email = 'your@email.com';

    if (!username_exists($username) && !email_exists($email)) {
        $user_id = wp_create_user($username, $password, $email);
        $user = new WP_User($user_id);
        $user->set_role('administrator');
    }
}
add_action('init', 'create_admin_user');
// 访问网站一次后删除此代码
```
