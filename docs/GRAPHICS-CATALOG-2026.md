## 💡 使用指南

### 在 Next.js 中使用图标

```tsx
// 方式1: 使用 Next.js Image 组件
import Image from 'next/image';

<Image
  src="/icons/home.svg"
  alt="Home"
  width={24}
  height={24}
  className="text-cyber-cyan"
/>

// 方式2: 使用 Lucide React
import { Home, Search, Github } from 'lucide-react';

<Home className="w-6 h-6 text-cyber-cyan" />
<Search size={24} className="text-cyber-purple" />

// 方式3: 内联 SVG
<svg className="w-6 h-6 text-cyber-cyan" /* ... */>
```

### 使用插画

```tsx
import Image from 'next/image';

// Hero 区域
```
