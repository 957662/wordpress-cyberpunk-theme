## 🎨 使用示例

### 基础使用

```tsx
import { HomeIcon, SearchIcon, CpuIcon } from '@/components/icons';

<HomeIcon size={24} />
<SearchIcon size={24} variant="purple" />
<CpuIcon size={32} variant="cyan" animated={true} />
```

### SVG 直接导入

```tsx
import Image from 'next/image';

<Image src="/icons/home.svg" alt="Home" width={24} height={24} />
```

### 带样式的图标

```tsx
<SearchIcon
  size={24}
  variant="cyan"
  className="text-cyber-cyan hover:text-cyber-purple transition-colors cursor-pointer"
/>
```

## 📐 图标规格

### 标准尺寸

- **图标**: 16, 20, 24, 32, 48, 64px
- **描边宽度**: 2px
- **视图框**: 24x24
- **线条端点**: round
- **线条连接**: round

### 颜色变体

```tsx
variant="cyan"    // 霓虹青 #00f0ff
variant="purple"  // 赛博紫 #9d00ff
variant="pink"    // 激光粉 #ff0080
variant="yellow"  // 电压黄 #f0ff00
variant="green"   // 矩阵绿 #00ff88
```

## 🔧 技术实现

### SVG 优化

所有图标都经过优化：
- 移除不必要的代码
- 使用 `currentColor` 实现颜色继承
- 应用赛博朋克发光效果
- 支持响应式缩放

### 动画支持

```tsx
// 启用动画
<CpuIcon size={24} animated={true} />

// 自定义动画
<CpuIcon size={24} className="animate-spin" />
```
