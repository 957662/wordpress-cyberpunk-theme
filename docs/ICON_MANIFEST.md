所有图标支持以下赛博朋克配色：

```css
/* 霓虹色系 */
--cyber-cyan:    #00f0ff;  /* 霓虹青 - 默认色 */

--cyber-pink:    #ff0080;  /* 激光粉 */
```

| `green` | 成功、在线、完成 | 成功、在线、完成状态 |

## 💡 使用示例

### 基础用法

```tsx
import { SearchIcon, BlogIcon, CpuIcon } from '@/components/icons';

// 默认使用
<SearchIcon />

// 自定义尺寸
<BlogIcon size={32} />

// 颜色变体
<CpuIcon size={48} variant="purple" />

// 组合使用
<SearchIcon size={24} variant="cyan" className="hover:scale-110" />
```

### 按钮图标

```tsx
<button className="flex items-center gap-2 px-4 py-2">
  <DownloadIcon size={20} variant="cyan" />
  <span>下载资源</span>
</button>
```

### 导航图标

```tsx
<nav className="flex gap-6">
  <a href="/" className="flex items-center gap-2">
    <HomeIcon size={20} variant="cyan" />
    <span>首页</span>
  </a>
  <a href="/blog" className="flex items-center gap-2">
    <BlogIcon size={20} variant="purple" />
    <span>博客</span>
  </a>
</nav>
```

### 状态图标

```tsx
{status === 'online' ? (
  <OnlineIcon size={16} variant="green" animated={true} />
) : (
  <OfflineIcon size={16} variant="pink" />
)}
```

### 社交链接

```tsx
<div className="flex gap-4">
  <a href="https://github.com/user" aria-label="GitHub">
    <GitHubIcon size={24} />
  </a>
  <a href="https://twitter.com/user" aria-label="Twitter">
    <TwitterIcon size={24} variant="cyan" />
  </a>
</div>
```

## ⚡ 动画效果

### 支持的动画类型

```tsx
// 脉冲动画（常用于在线状态）
<OnlineIcon size={16} animated={true} />

// 旋转动画（加载状态）
<LoadingIcon size={24} animated={true} />

// 悬停动画（通过 Tailwind）
<SearchIcon
  size={24}
  className="hover:scale-110 hover:rotate-12 transition-all"
/>
```

### 动画建议

- ✅ **脉冲**: 在线状态、加载中
- ✅ **旋转**: 加载状态、同步中
- ✅ **悬停**: 所有交互图标
- ❌ **避免**: 过度使用动画影响性能

## ♿ 可访问性

```tsx

  size={24}
  role="img"
  aria-label="搜索"
/>

  size={20}
  role="button"
  aria-label="收藏文章"
  aria-pressed={isFavorited}
/>
```

### 键盘导航

```tsx
<button
  className="focus:ring-2 focus:ring-cyber-cyan"
  onClick={handleAction}
  aria-label="关闭"
>
  <CloseIcon size={20} />
</button>
```
