## 🎨 设计规范

### 配色方案

```css
/* 主色调 */
--cyber-dark: #0a0a0f      /* 深空黑 - 背景 */
--cyber-cyan: #00f0ff      /* 霓虹青 - 主要强调色 */
--cyber-purple: #9d00ff    /* 赛博紫 - 次要强调色 */
--cyber-pink: #ff0080      /* 激光粉 - 强调色 */

/* 辅助色 */
--cyber-green: #00ff88     /* 赛博绿 - 成功状态 */
--cyber-yellow: #f0ff00    /* 电压黄 - 警告状态 */
--cyber-muted: #1a1a2e     /* 深空蓝 - 次级背景 */
--cyber-card: #16162a      /* 卡片背景 */
--cyber-border: #2a2a4a    /* 边框颜色 */
```

### 字体系统

```css
/* 无衬线字体 - 正文 */
font-family: 'Inter', system-ui, sans-serif;

/* 展示字体 - 标题 */
font-family: 'Orbitron', sans-serif;

/* 等宽字体 - 代码 */
font-family: 'JetBrains Mono', monospace;
```

### 设计元素

- **故障效果 (Glitch)**: 用于标题和强调文本
- **扫描线 (Scanlines)**: 覆盖整个页面的复古效果
- **霓虹发光 (Neon Glow)**: 文字和边框发光效果
- **全息投影 (Hologram)**: 卡片和面板的背景效果
- **粒子动画 (Particles)**: 背景粒子效果

## 📝 设计规范文档

### 图标使用规范

```tsx
// 使用 Lucide Icons
import { Search, Menu, User } from 'lucide-react';

// 使用自定义图标
import { CyberIcon } from '@/components/icons';

// 设置图标样式
<Search className="w-5 h-5 text-cyber-cyan" />
```

### 颜色使用规范

```tsx
// 主要强调色

// 次要强调色

// 边框颜色

className="bg-cyber-dark"
```
