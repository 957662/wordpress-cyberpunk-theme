## 🚀 快速开始

### 最简单的方式（推荐）

```bash
# 1. 进入项目目录
cd cyberpress-platform

# 2. 运行启动脚本
./start-dev.sh

# 3. 访问应用
# 前端: http://localhost:3000
# 后端: http://localhost:8000
# API文档: http://localhost:8000/docs
```

### 停止服务

```bash
./stop-dev.sh
```

### 重启服务

```bash
./restart-dev.sh
```
