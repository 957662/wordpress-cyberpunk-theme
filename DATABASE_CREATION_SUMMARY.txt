🎯 核心功能亮点

【1. 搜索优化系统】

  特性:
  ✓ 全文搜索支持中文（N-gram分词）
  ✓ 最小词长支持（2字符）
  ✓ 搜索历史自动记录
  ✓ 热门关键词智能统计
  ✓ 趋势分数自动计算
  ✓ 搜索统计可视化

  使用:
  • 搜索: SELECT * FROM posts WHERE MATCH(title, content, excerpt) 
           AGAINST('关键词' IN NATURAL LANGUAGE MODE);

  • 记录: CALL sp_record_search('关键词', 结果数, '类型', 用户ID, 'IP');

  • 热门: SELECT * FROM popular_keywords ORDER BY search_count DESC;

【2. 性能监控系统】

  特性:
  ✓ 慢查询自动记录和分析
  ✓ 表性能趋势追踪
  ✓ 索引使用率统计
  ✓ 碎片化程度检测
  ✓ 自动优化建议
  ✓ 性能报告生成

  使用:
  • 慢查询: SELECT * FROM v_slow_query_summary LIMIT 10;

  • 索引: SELECT * FROM v_unused_indexes;

  • 趋势: SELECT * FROM v_table_performance_trend;

  • 报告: CALL sp_generate_performance_report();

【3. 自动化运维】

  特性:
  ✓ 一键初始化数据库
  ✓ 智能备份和恢复
  ✓ 实时性能监控
  ✓ 全面健康检查
  ✓ 彩色输出提示
  ✓ 详细日志记录

  使用:
  • 初始化: ./scripts/db-init.sh

  • 恢复:   ./scripts/db-restore.sh backup.sql.gz

  • 监控:   ./scripts/db-monitor.sh --report

  • 检查:   ./scripts/db-health-check.sh
