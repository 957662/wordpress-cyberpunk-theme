## 🛠️ 技术栈

### 前端
- **框架**: Next.js 14 (App Router)
- **UI**: React 18, TypeScript 5.4
- **样式**: Tailwind CSS 3.4
- **动画**: Framer Motion 11
- **图标**: Lucide Icons
- **状态**: Zustand 4.5
- **请求**: TanStack Query 5.28

### 后端
- **数据库**: MySQL 8.0+ / MariaDB 10.6+
- **认证**: JWT
- **API**: Next.js API Routes
- **ORM**: Prisma (计划)

### 开发工具
- **语言**: TypeScript 5.4
- **包管理**: npm / yarn / pnpm
- **代码检查**: ESLint
- **格式化**: Prettier
- **Git**: Hooks
